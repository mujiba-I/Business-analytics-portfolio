Sales Performance Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard built using Power BI. The dashboard helps analyze sales performance through key metrics and visualizations, enabling users to identify trends and make data-driven business decisions.

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor key business metrics.
- Identify top-performing categories and regions.
- Visualize sales trends through interactive dashboards.

## 📊 Dashboard Features
- KPI Cards (Sales, Profit, Quantity, Discount)
- Bar Charts
- Pie & Donut Charts
- Funnel Chart
- Map Visualization
- Interactive Slicers and Filters

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Power Query

## 📁 Files Included
- Sales Performance Dashboard.pbix
- Dataset
- Dashboard Screenshot
- README.md

## 💡 Key Insights
- Monitored overall sales, profit, quantity, and discount.
- Compared sales across different regions and categories.
- Identified business trends using interactive visualizations.
- Enabled dynamic filtering for better data exploration.
